# Bookshop Sample App

## Prerequisites:

* Node.js v8
* git
* sqlite3 (pre-installed on Macs and Linux)

## Setup

```sh
git clone https://github.wdf.sap.corp/caps/bookshop.git
cd bookshop
npm install
cds deploy
```

## Run

```sh
cds run
```
